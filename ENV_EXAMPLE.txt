# RETURN MVP 환경 변수 설정 가이드

이 파일을 복사하여 backend/.env 파일을 만드세요.

## backend/.env 파일 생성

DATABASE_URL="file:./dev.db"
PORT=3000
NODE_ENV=development

# Kakao Maps API 키 (필수!)
# https://developers.kakao.com/ 에서 발급
KAKAO_REST_API_KEY=your_kakao_rest_api_key_here
KAKAO_ADMIN_KEY=your_kakao_admin_key_here

# Gmail 이메일 설정 (선택)
# 없으면 콘솔에만 출력됩니다
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-gmail-app-password

FRONTEND_URL=http://localhost:5173
USE_REAL_PAYMENT=false

---

## frontend/index.html 수정

11번 줄의 YOUR_JAVASCRIPT_KEY_HERE를 실제 Kakao JavaScript 키로 교체하세요.

<script src="//dapi.kakao.com/v2/maps/sdk.js?appkey=YOUR_JAVASCRIPT_KEY_HERE&libraries=services"></script>

---

## 발급 방법

### Kakao API 키
1. https://developers.kakao.com/ 접속
2. 애플리케이션 생성
3. REST API 키, JavaScript 키 복사

### Gmail 앱 비밀번호
1. Google 계정 → 보안 → 2단계 인증 활성화
2. https://myaccount.google.com/apppasswords
3. 앱 비밀번호 생성

자세한 내용은 docs/SETUP_GUIDE.md 참조
